# Full Stack Development Internship

This repository contains the projects completed as part of the DecodeLabs Full Stack Development Internship. Each phase focuses on a fundamental concept of full stack development, progressing from frontend development to backend APIs, database integration, and frontend-backend communication.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+, Async/Await, Fetch API)
- Node.js
- Express.js
- Python
- FastAPI
- SQLite
- SQLAlchemy

---

## Phase 1 – Frontend Development

**Objective**

Develop a responsive frontend interface using HTML, CSS, and JavaScript.

**Topics Covered**

- HTML structure
- CSS styling
- Responsive design
- JavaScript fundamentals
- User interface development

---

## Phase 2 – Backend API Development

**Objective**

Build RESTful APIs to handle application logic.

**Features**

- GET API
- POST API
- PUT API
- DELETE API
- Request validation
- JSON responses

**Technology**

- Python
- FastAPI

---

## Phase 3 – Database Integration

**Objective**

Connect the backend with a database for persistent data storage.

**Features**

- Database schema
- CRUD operations
- SQLAlchemy ORM
- SQLite database
- Data validation

**Technology**

- Python
- FastAPI
- SQLAlchemy
- SQLite

## Phase 4 – Frontend & Backend Integration

**Objective**

Integrate the frontend with backend APIs using asynchronous JavaScript.
